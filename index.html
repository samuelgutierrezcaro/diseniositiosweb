PRUEBAS DE SOFTWARE

Las pruebas son muy importantes en el desarrollo del software, la fase de pruebas es de las mas costosas, pero es una buena inversión a largo plazo, a través de estas pruebas, validamos que los procesos que debe realizar el programa cumpla con lo esperado por el usuario, hace parte del control de calidad, los resultados de las pruebas son analizados, realizando la evaluación de alguna característica.

Las pruebas de software, se definen como una actividad en la cual se ejecuta un sistema, parcial o totalmente, bajo unas circunstancias específicas, las pruebas de software han evolucionada paulatinamente, pasando de ser una metodología convencional, a una metodología ágil


OBJETIVOS
  • Objetivo General: Explicar los tipos de pruebas realizados en el proceso de desarrollo de software
  • Objetivo 1: Definir el tipo de pruebas funcionales y no funcionales
  • Objetivo 2: Enunciar el tipo de pruebas estructurales.
  • Objetivo 3: Indicar el tipo de pruebas asociadas al cambio
