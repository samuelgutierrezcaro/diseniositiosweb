Objetivo general: El OVI tiene como objetivo explicar los tipos de pruebas realizados en el proceso de desarrollo de software.
Objetivo especifico 1: Darle a conocer al estudiante el tipo de pruebas funcionales y no funcionales
Objetivo especifico 2: Explicarle al estudiante el tipo de pruebas estructurales.
Objetivo especifico 3: Mostrarle al estudiante el tipo de pruebas asociadas al cambio.
Las pruebas de software (en inglés software testing) son las investigaciones empíricas y técnicas cuyo objetivo es proporcionar información objetiva e independiente sobre la calidad del producto a la parte interesada o stakeholder.
